Converting files

Xena can convert an input file into an openly specified format, such as ODF or PNG.

Convert is different to Normalisation as it doesn't produce a Xena file; the input file is converted to an open format and placed in the Destination directory (see Setting Preferences).

If the input file is already in an openly specified format, it is copied to the Destination directory.

Xena can convert single files or entire directories of files.
